info_spider
===

##### 一个舆情搜索爬虫,这是一个练手项目，很多功能还需要完善！
---
## info_spider是什么?

* 可通过`自定义`过滤关键字，eg:爆炸、毒品等
* 可通过`自定义`添加贴吧 eg:李毅吧、胥渡吧等
* 可进行筛选导出`html`搜索报告
* 每个模块单独生成`本地json文件资源`
* 将加入微信公众号文章、新浪微博、以及论坛site等模块

## info_spider有哪些功能？

* 爬取`百度贴吧`模块，无贴吧数量限制
* 爬取`大江论坛`模块，同时爬取十个论坛
* 爬取`天涯论坛`模块，同时十个论坛
* 方便的`导入导出`功能
    *  导出为一个`html格式`的文件

## 用法
* #### 下载项目
``` javascript
git clone https://github.com/bbkali/info_spider.git
```
* #### 安装依赖
``` javascript
pip install -r requirements.txt
```
* #### 添加配置
``` javascript
vim config.ini|vim config_sample.ini //自行添加配置信息
```
* #### 启动爬虫
``` javascript
python run.py
```
